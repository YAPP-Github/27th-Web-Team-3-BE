# API-016 참여자 조회

> `GET /api/v1/retrospects/{retrospectId}/participants`

---

## Flow

```mermaid
sequenceDiagram
    participant C as Client
    participant S as Server
    participant DB as Database

    C->>S: GET /retrospects/{id}/participants
    S->>S: accessToken 검증
    S->>DB: 팀 멤버 확인
    alt 멤버 권한
        S->>DB: 참여자 목록 조회
        S->>DB: 질문 목록 조회
        S-->>C: totalCount, participants, questions
    else 권한 없음
        S-->>C: TEAM4031 에러
    end
```

---

## Quick Reference

| 항목 | 값 |
|------|-----|
| **Method** | GET |
| **Auth** | accessToken (Bearer) |
| **Body** | 없음 |

## 정렬 순서

| 필드 | 기준 | 순서 |
|------|------|------|
| participants | 참석 등록일 | 오름차순 |

---

## Error Codes

| Code | Status | 설명 |
|------|--------|------|
| COMMON400 | 400 | 잘못된 retrospectId |
| AUTH4001 | 401 | 인증 실패 |
| TEAM4031 | 403 | 팀 멤버가 아님 |
| RETRO4041 | 404 | 존재하지 않는 회고 |
