# API-011 팀 회고 목록

> `GET /api/v1/retro-rooms/{retroRoomId}/retrospects`

---

## Flow

```mermaid
sequenceDiagram
    participant C as Client
    participant S as Server
    participant DB as Database

    C->>S: GET /retro-rooms/{id}/retrospects
    S->>S: accessToken 검증
    S->>DB: 멤버 권한 확인
    alt 멤버 권한
        S->>DB: 회고 목록 조회
        DB-->>S: 회고 리스트 (최신순)
        S-->>C: 회고 목록
    else 권한 없음
        S-->>C: RETRO4031 에러
    end
```

---

## Quick Reference

| 항목 | 값 |
|------|-----|
| **Method** | GET |
| **Auth** | accessToken (Bearer) |
| **Body** | 없음 |

## retrospectMethod Enum

| 값 | 설명 |
|----|------|
| KPT | Keep-Problem-Try |
| FOUR_L | 4L (Liked, Learned, Lacked, Longed For) |
| FIVE_F | 5F (Facts, Feelings, Findings, Future, Feedback) |
| PMI | Plus-Minus-Interesting |
| FREE | 자유 형식 |

## Error Codes

| Code | Status | 설명 |
|------|--------|------|
| AUTH4001 | 401 | 인증 실패 |
| RETRO4031 | 403 | 멤버가 아님 |
| RETRO4041 | 404 | 존재하지 않는 회고방 |
