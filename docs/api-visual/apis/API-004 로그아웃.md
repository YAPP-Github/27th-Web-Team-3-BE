# API-004 로그아웃

> `POST /api/v1/auth/logout`

---

## Flow

```mermaid
sequenceDiagram
    participant C as Client
    participant S as Server
    participant DB as Database

    C->>S: POST /auth/logout
    Note right of C: accessToken (Header), refreshToken (Body)
    S->>S: accessToken 검증
    alt 인증 성공
        S->>DB: refreshToken 무효화
        DB-->>S: 삭제 완료
        S-->>C: 로그아웃 성공
    else 인증 실패
        S-->>C: AUTH4001 에러
    end
```

---

## Quick Reference

| 항목 | 값 |
|------|-----|
| **Method** | POST |
| **Auth** | accessToken (Bearer) |
| **Content-Type** | application/json |

---

## Request Body

```json
{
  "refreshToken": "service_refresh_token_xxx"
}
```

> [!tip] 다중 기기 지원
> refreshToken을 명시적으로 전달하여 특정 기기만 로그아웃할 수 있습니다.

---

## Response

```json
{
  "isSuccess": true,
  "code": "COMMON200",
  "message": "로그아웃이 성공적으로 처리되었습니다.",
  "result": null
}
```

---

## Error Codes

| Code | Status | 설명 |
|------|--------|------|
| AUTH4001 | 401 | 인증 정보 유효하지 않음 |
| AUTH4003 | 400 | 이미 로그아웃/유효하지 않은 토큰 |
